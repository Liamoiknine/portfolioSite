# Portfolio Site

My personal portfolio website built with React and Vite.

## What's On The Site

The site has three main sections: an intro page with some background about me and what I'm up to, a projects page showing active work and past projects, and an experience page with professional and personal stuff.

## Tech Stack

- React
- Vite
- Vanta.js for the animated background
- Typewriter effect for text animations
